# RestoreAssist YouTube Channel — Dual-Use Curriculum Structure
## 90-Day Content Calendar + CPD Pathway Design

**Source:** CEO Board DELTA Strategy, 2026-03-29
**Owner:** Phill McGurk
**Sprint:** Month 1 planning

---

## Design Principles

1. **Dual-use from day one** — every video is both a YouTube watch and a CPD learning unit
2. **Progressive competency build** — each module builds on the last; no orphaned content
3. **Australian-first framing** — every lesson leads with Australian regulatory context
4. **No product CTA in Months 1–3** — pure educational authority building
5. **5-hour weekly production cap** — 2 videos per week maximum (batched 4-at-a-time)

---

## LMS Integration Plan (Month 6+)

When the training platform is built, each video module maps to a structured learning unit:

```
Module → Lesson Plan:
  1. Pre-reading (linked article or PDF)
  2. Video content (YouTube embed)
  3. Knowledge check (3–5 MCQs)
  4. Practical application task (optional for CPD accreditation)
  5. CPD certificate trigger (on completion + pass)
```

**Recommended LMS platforms** for Month 6 build:
- **Teachable** — fastest to launch, limited CPD metadata
- **Thinkific** — better for B2B (NRPG/CARSI partnerships)
- **LearnDash** (WordPress) — most flexible CPD certificate control
- **Custom (RestoreAssist-integrated)** — highest retention, requires ~120 hours build

---

## Production Schedule Constraints

- **Weekly time cap:** 5 hours (non-negotiable)
- **Batch recording:** Film 4 videos in one session, publish weekly
- **Script-to-publish pipeline:** Script (AI-assisted) → Record → Edit (basic cuts only) → Thumbnail → Publish → Post in Facebook groups
- **Tools:** Scripts in `docs/youtube-scripts/`, thumbnails via Canva, editing via CapCut/DaVinci
- **Monetisation unlock:** YouTube Partner Programme requires 1,000 subscribers + 4,000 watch hours → target Month 5 based on conservative projections
